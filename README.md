# Diabetess Prediction using Logistic Regression

A medical data science project that predicts the risk of a patient developing diabetes risk using patient data from the Pima Indians Diabetes dataset.

## Overview
This project demonstrates the full data-science workflow:
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature scaling and model training  
- Evaluation with ROC/AUC metrics and feature interpretation  

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- **Accuracy:** 76.6%  
- **AUC:** 82.3%  
- **Top Features:** Glucose, BMI, Age
